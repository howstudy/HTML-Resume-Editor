# HTML-简历编辑器
一个无需任何外部依赖的纯前端工具，可以导入任意 HTML 简历 → 可视化编辑所有文字字段 → 导出 HTML/PDF。
## ✨ 为什么做这个项目？
在求职过程中，我尝试用网页端大模型生成简历，但生成后的 HTML 简历如果需要微调，只能用记事本或 IDE 手动修改——既繁琐又不直观。
如果能像编辑在线表单一样，直接点选修改 HTML 简历的每一段文字，那该多方便？
于是，我借助 vibe coding 工具，并运用原生 JavaScript 技术，独立开发了这个「可视化 HTML 简历编辑器」。
## 🚀 快速上手
1. 下载或克隆本仓库  
2. 双击 `简历编辑器.html` 直接运行（无需服务器）  
3. 点击「导入 HTML」上传你的简历文件 → 左侧所有文字都会变成可编辑的输入框  
4. 随意修改内容、拖拽排序 → 右侧实时预览  
5. 点击「导出 HTML」或「导出 PDF」完成新简历
